Amusement #1 - RawAccel

RawAccel is a program that changes mouse sensitivity depending on how fast the mouse is moving. This is done by feeding inputs through a custom input driver that manipulates raw input. RawAccel is the best mouse acceleration program due to its compliance with anti-cheats as well as its customizability.
